# Secure Password Generator

This is a beginner-friendly Python project that generates random secure passwords.

The program creates strong passwords using:
- lowercase letters
- uppercase letters
- numbers
- special symbols

Users can choose the password length, and the program generates a random password instantly.

---

# Features

- Random password generation
- User-defined password length
- Uses uppercase and lowercase letters
- Includes numbers and symbols
- Beginner-friendly Python project

---

# Technologies Used

- Python
- Random Module
- Strings
- Loops
- User Input

---

# How to Run

1. Install Python
2. Save the file as `password_generator.py`
3. Run the program
