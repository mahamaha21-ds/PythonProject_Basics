# Calculator With History

This is a simple Python calculator project created for beginners.

The calculator can:
- Add numbers
- Subtract numbers
- Multiply numbers
- Divide numbers

The program also stores calculation history in a text file.

---

# Features

- Menu-based calculator
- Uses Python functions
- Uses return statements
- Handles invalid input
- Handles divide by zero error
- Saves calculation history in `calculator.txt`

---

# Technologies Used

- Python
- File Handling
- Functions
- Exception Handling

---

# How to Run

1. Install Python
2. Save the code as `calculator.py`
3. Run the program

```bash
python calculator.py
